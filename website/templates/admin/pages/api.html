API Documentation

Base URL

HTTP: http://127.0.0.1:5000/api/v1.0/magnews

HTTPS: https://127.0.0.1:5000/api/v1.0/magnews

Supported Methods: GET

Request Parameters

key = [key] (REQUIRED)

  • key - Your API Key.

ID = [ID] (OPTIONAL)

  • ID - Post ID.

Example Request

All Posts: https://127.0.0.1:5000/api/v1.0/magnews?key=API_KEY

Posts by ID: https://127.0.0.1:5000/api/v1.0/magnews/ID?key=API_KEY
